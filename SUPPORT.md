# Support

## Where To Get Help

Use the support path that matches what you need:

- GitHub Discussions: questions, walkthrough help, training ideas, and badge-track suggestions
- GitHub Issues: reproducible bugs, broken scripts, workflow problems, or documentation errors
- Pull Requests: proposed fixes or improvements that are ready for review

## Before Opening An Issue

Please check these first:

- `README.md` for setup and lab overview
- `docs/guides/` for testing, methodology, and Windows workflow guidance
- `CONTRIBUTING.md` for branch, PR, and validation expectations

## Badge And Completion Help

If you want to earn the project completion badge:

1. Complete the lab work and keep evidence in `reports/` or your write-up.
2. Open the `Actions` tab in GitHub.
3. Run the `award-completion-badge` workflow on `main`.

Available tracks:

- `Full Lab Completion`
- `Exploitation Track`
- `Reporting Track`
- `Recon Track`

## Safety Reminder

This repository is intentionally vulnerable for local learning. Use it only in isolated environments you control.
