# Wireless Assessment

This folder is reserved for authorized, passive wireless lab notes and evidence.

Use it for:

- approved scope notes
- lab observation logs
- channel and beacon mapping
- packet-capture notes
- screenshots and timestamps
- defensive review findings

Do not use this folder for disruptive testing, deauthentication activity, or any unauthorized wireless interference.

## Suggested Structure

- `AUTHORIZED_SCOPE_TEMPLATE.md`
- `PASSIVE_NOTES_TEMPLATE.md`
- `pcaps/`
- `screenshots/`
- `reports/`

## Passive Tooling Notes

These tools may be relevant for passive, authorized lab work:

- `Aircrack-ng` for monitor-mode capture review
- `Bettercap` for local lab observation and host visibility
- `Scapy` for packet inspection and offline packet handling
- `Wireshark` for packet capture analysis
- `Kismet` for passive wireless discovery and logging

Any use must stay within your own lab or clearly authorized environment.
